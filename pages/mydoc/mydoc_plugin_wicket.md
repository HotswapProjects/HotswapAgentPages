Support for [Apache Wicket](https://wicket.apache.org/) Wicket is an open source, component oriented, serverside, Java web application framework. 

To create a Wicket project, go to https://wicket.apache.org/start/quickstart.html

#### Features
* Localizer caches are cleared whenever a property file is changed
